## 技术 - Record

> Record<K, T> 构造具有给定类型 T 的一组属性是 K 的类型。在将一个类型的属性映射到另一个类型的属性时，Record 非常方便

```typescript
// bad
interface User {
  name: string
}
const users: { [key: string]: User } = {}

// good
interface User {
  name: string
}
const users: Record<string, user> = {}
```
