## 技术 - vscode-file-nesting-config

[vscode-file-nesting-config](https://github.com/antfu/vscode-file-nesting-config) 是一个 `VSCode` 配置，它可以使你的文件以树形结构呈现，这样一些不常用的文件可以隐藏在结构里了，使项目看起来更清晰。
