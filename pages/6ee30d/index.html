# 房贷试算器 - TypeScript 版

基于前面已学习的内容，对 2 年前的 JavaScript 版房贷试算项目，进行 TypeScript 改造。希望由具体的实战项目，能够强化已学习的知识点，以及在项目中的应用！

项目主要通过 Webpack 进行打包，项目代码 (opens new window)

构建 mortgageCalculator 类的 JavaScript 文件：


npm run build-lib

运行本地示例：项目示例 (opens new window)


npm run dev

构建打包示例代码到 dist：


npm run build
